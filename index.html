Projets

Pendant ces 5 projets nous avons obtenus divers compétences, et avons appris les bases de plusieurs langages de programmation.

Projet 1

Dans ce projet nous avons appris à nous exprimer à l'oral ainsi qu'à préparer un powerpoint bien construit et propre.

Projet 2

Dans ce projet nous avons appris à créer un serveur ssh en local ou sur le web. à partir d'un terminal ubuntu, de mysql, de php et de logiciel comme roundcube et ispconfig.

Projet 3

Dans ce projet nous avons appris à créer des algorithme sur larp ou bien python. Nous avons coder un morpion et un pierre, feuille, ciseau avec VIM sous Linux.

Projet 4

Dans ce Projet nous avons appris à créer un site Web de A à Z, en HTML et CSS. Nous avons tout de même utiliser du JavaScript afin d'implanter un mode nuit/jour sur le site.

Projet 5

Dans ce projet nous avon appris à créer un site Web, en HTML, CSS et JavaScript, majoritairement grâce à Materialize.
